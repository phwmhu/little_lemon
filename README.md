Little Lemon Project

Little Lemon is a web application designed to streamline the table booking process for the Little Lemon restaurant. Customers can easily make reservations, view special offers, and access important restaurant information.

Features
User-friendly table booking form
Real-time availability check
Special offers and promotions
Accessibility features
Responsive design
Unit tests for reliability
Seamless user experience
